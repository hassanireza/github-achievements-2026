We show you the legitimate path that earns the exact same badge instead.

## ⚠️ About the "two accounts" trick

You will see this "tip" everywhere: make a second GitHub account, co-author a commit between the two, instantly get `Pair Extraordinaire`. We're not including it as a tactic, and here's the actual reasoning, not just a refusal:

> GitHub's Terms of Service (§3, Account Requirements) cap you at **one free personal account per person**. Using a second account to game an achievement is precisely the "manipulate platform features" pattern that gets accounts suspended — and it's easy to detect via shared IP, device fingerprint, and commit timing correlation. You'd be risking your real, primary account — every repo, every contribution streak, every org membership — for a cosmetic badge you can get just as fast by asking one real person to pair with you for ten minutes.

The guide shows the legitimate version: find one real collaborator (a friend, a study-group peer, anyone), use git's actual `Co-authored-by:` trailer, get the same badge, keep your account. Full comparison is in the **Risk Check** section of the page.
